## 使用建议

### 初次使用建议操作流程

1. 启动服务后，打开 http://localhost:3000
2. 保持默认参数，点击"运行回测"按钮
3. 观察K线图、权益曲线和统计指标
4. 尝试调整快线周期到30，再次运行回测，对比结果
5. 在右侧交易面板尝试开仓和平仓操作
6. 查看持仓和订单历史

### 参数调优建议

- 快线周期（20）：数值越小越灵敏，但假信号越多
- 慢线周期（50）：数值越大趋势越稳定，但反应越慢
- 最佳配比通常是慢线 = 2-3倍快线
- ATR倍数：2.0是常用值，增大更保守，减小更激进

### 策略适用场景

均线交叉策略适合：
- ✓ 趋势明显的市场
- ✓ 中长期持仓
- ✓ 波动较大的标的

不适合：
- ✗ 震荡盘整市场
- ✗ 日内短线交易
- ✗ 波动率极低的标的

## 常见问题快速解决

### Q1: 后端启动失败，提示找不到模块
**解决**：确保已运行 `backend/setup.bat` 初始化虚拟环境

### Q2: 前端页面空白
**解决**：
1. 确认后端服务已启动（访问 http://localhost:8000 测试）
2. 打开浏览器F12开发者工具查看Console错误
3. 检查Network标签，确认API请求成功

### Q3: 图表不显示
**解决**：点击"运行回测"按钮加载数据，初次打开需要手动触发

### Q4: 端口被占用
**解决**：
- 8000端口：修改 backend/main.py 最后一行的 port 参数
- 3000端口：修改 frontend/package.json 的 dev 脚本为 `next dev -p 3001`

### Q5: C盘空间不足
**解决**：本项目已配置虚拟环境在项目目录下，不占用C盘系统空间
