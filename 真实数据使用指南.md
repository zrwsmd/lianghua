# 真实数据使用指南

系统已接入AkShare数据源，支持获取A股、黄金、白银等真实历史数据。

## 支持的资产类型

### 1. A股股票

在"交易标的"输入框中输入6位股票代码：

热门股票代码示例：
- 600000 - 浦发银行
- 600519 - 贵州茅台
- 000001 - 平安银行
- 000002 - 万科A
- 600036 - 招商银行
- 601318 - 中国平安
- 000858 - 五粮液
- 002594 - 比亚迪

注意：
- 只需输入6位数字代码，不需要加SH或SZ前缀
- 目前只支持日线数据（选择时间周期时选"1d"）

### 2. 黄金/白银

在"交易标的"输入框中输入：

贵金属代码：
- Au99.99 - 上海黄金交易所现货黄金
- Ag99.99 - 上海黄金交易所现货白银
- Au(T+D) - 黄金延期交收
- Ag(T+D) - 白银延期交收

### 3. 加密货币（模拟数据）

- BTCUSDT - 比特币
- ETHUSDT - 以太坊
- 其他加密货币仍使用模拟数据

## 使用步骤

### 步骤1：选择资产

在左侧策略参数面板：
1. 在"交易标的"输入框中输入股票代码或贵金属代码
2. 例如：输入 600519（贵州茅台）

### 步骤2：设置回测参数

1. 时间周期：选择"1d"（日线，真实数据只支持日线）
2. 开始日期：例如 2023-01-01
3. 结束日期：例如 2024-12-31
4. 策略参数：根据需要调整均线周期

### 步骤3：运行回测

点击"运行回测"按钮，系统会：
1. 自动识别资产类型（A股/黄金/加密货币）
2. 从AkShare获取真实历史数据
3. 如果获取失败，自动降级为模拟数据
4. 显示K线图和回测结果

## 注意事项

1. 数据来源：所有真实数据来自AkShare开源库，完全免费

2. 数据范围：
   - A股：支持所有A股股票的历史数据
   - 黄金白银：上海黄金交易所数据
   - 时间跨度：根据AkShare的数据覆盖范围

3. 数据更新：
   - A股数据：每日收盘后更新
   - 不是实时数据，有一定延迟

4. 降级机制：
   - 如果真实数据获取失败（网络问题、停牌等）
   - 系统会自动使用模拟数据
   - 确保回测能继续进行

5. 分钟级数据：
   - 目前真实数据只支持日线（1d）
   - 如果选择5m、15m等分钟周期，会使用模拟数据
   - 未来版本会扩展支持分钟数据

## 数据质量说明

1. A股数据：
   - 使用前复权价格（已调整除权除息）
   - 包含开高低收和成交量
   - 数据准确性高

2. 黄金数据：
   - 上海黄金交易所官方数据
   - 单位：元/克
   - 包含完整的OHLCV数据

3. 模拟数据：
   - 使用随机游走模型生成
   - 仅供学习和测试策略逻辑
   - 不代表真实市场行为

## 常见问题

Q1: 为什么有些股票代码查不到数据？
A: 可能是停牌、退市或代码输入错误。系统会自动降级为模拟数据。

Q2: 可以获取美股数据吗？
A: 目前只支持A股和国内贵金属。美股数据需要其他API，未来版本会考虑添加。

Q3: 如何获取最新的实时数据？
A: 当前版本主要用于历史回测，实时数据需要专门的行情接口。

Q4: 数据获取很慢怎么办？
A: 首次获取数据需要从网络下载，请耐心等待。后续可以添加本地缓存功能。

Q5: 可以回测期货吗？
A: 目前期货接口还在开发中，暂时使用模拟数据。

## 技术说明

数据提供器代码位置：`backend/data/data_provider.py`

资产识别逻辑：
- 6位纯数字 → A股
- Au99.99 / Ag99.99 → 贵金属
- 其他 → 加密货币（模拟）

如需扩展其他数据源，可以在DataProvider类中添加新的获取方法。
